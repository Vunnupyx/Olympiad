## Процесс сборки
Requires [Node.js](https://nodejs.org/) to run.

Install the dependencies and devDependencies

```sh
$ git clone https://github.com/Vunnupyx/Olympiad.git
```

Start the server.

```sh
$ cd server
$ npm install
$ npm run server // Starting the server
```

Start the client.

```sh
$ cd frontend
$ npm install
$ npm start
```
